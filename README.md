## 매뉴얼

1. [Discord Chat Exporter](https://github.com/Tyrrrz/DiscordChatExporter/releases/tag/2.25) 에서 ChatExporter를 다운 받는다.
2. ChatExporter의 자체 매뉴얼대로 채팅내역을 Export 한다.
* 주의 : Media Export 체크!!
3. Export 된 파일들로 이 Repository 에 PR을 넣는다.
